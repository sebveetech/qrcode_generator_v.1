# qrcode_generator_v.1
Modern QR Code Generator <br>
The QR Code Generator by STV is a simple and modern web-based tool that allows users to instantly create high-quality QR codes from text or URLs. It is designed with a clean interface and responsive layout, making it easy to use on both desktop and mobile devices.<br><br>
⚙️ Features<br>
✅ Generate QR codes from any text or URL<br>
📏 Multiple size options (200px to 400px)<br>
💾 Download QR codes as PNG images<br>
🎨 Clean and modern user interface<br>
📱 Fully responsive design (mobile-friendly)<br>
⚡ Fast and lightweight (no backend required)<br><br>
🚀 How to Use<br>
Enter your text or URL in the input box<br>
Select your preferred QR code size<br>
(Optional) Set a custom file name<br>
Click "Generate QR"<br>
Preview the generated QR code instantly<br>
Click "Download PNG" to save the image<br><br>
🔄 Clear Function<br>
Click the "Clear" button to:<br>
Remove the generated QR code<br>
Reset the input field
